# Jadu: Class 11 Study Assistant

A simple web application that uses AI to convert chapter text into revision materials. This tool is designed to help Class 11 students prepare for School Boards and JEE exams.

## Features
* **Formula Extraction:** Automatically identifies and formats mathematical formulas using LaTeX.
* **Summarization:** Provides bullet-point summaries of core chapter concepts.
* **Practice Questions:** Generates 3 MCQs based on the provided text.
* **Hidden Solutions:** Answers are hidden behind a toggle to allow for self-testing.

## Tech Stack
* **Language:** Python
* **Framework:** Streamlit
* **AI:** Google Gemini 2.5 Flash
